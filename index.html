Enter the email address to receive the user data report

ℹ️ How to use:

  1. Click "Test API Connection" to check if API is ready
  2. Wait for green checkmark to appear
  3. Enter the recipient email address
  4. Click "Send Email Notification"
  5. Email will contain all user data in a table format

Note: API endpoint is configured in Azure Static Web App settings.
